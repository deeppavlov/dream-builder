## Building

Requirements:
 - Node 16+
 - Python 3.9+
 - [pnpm](https://pnpm.io)
 - [poetry](https://python-poetry.org/)

To run a local development version at http://localhost:8080:
```
pnpm run dev
```

To run tests
```
pnpm test
```
